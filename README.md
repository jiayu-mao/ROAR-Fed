# ROAR-Fed

This repository contains the implementation for the ROAR-Fed framework.

## Status
Code is currently being organized and will be fully released.

## Reproducibility
All experiments in the paper are conducted under a unified pipeline with fixed Dirichlet partitions and multi-seed averaging.

## Paper
Details can be found in the associated manuscript: https://arxiv.org/abs/2412.03514.
